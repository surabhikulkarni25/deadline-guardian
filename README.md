# Deadline Guardian ⏳🛡️

An AI-powered productivity companion designed to help users **stop missing deadlines** and take meaningful action before it is too late.

Unlike traditional productivity tools that rely on passive reminders, **Deadline Guardian** actively assists users in planning, prioritizing, and completing tasks through intelligent risk analysis, adaptive planning, personalized AI guidance, and accountability-driven interventions.

---

## Problem Statement

Students, professionals, and entrepreneurs frequently miss deadlines, assignments, meetings, bill payments, interviews, and important commitments.

Most productivity applications only provide passive reminders, which are easy to ignore and often fail to drive real action.

**Deadline Guardian solves this by acting as an active AI productivity companion rather than just a reminder tool.**

---

## Solution Overview

Deadline Guardian analyzes tasks using multiple parameters such as:

* Deadline
* Importance
* Urgency
* Risk factor
* Task completion behavior
* User productivity patterns

Using this information, the system proactively guides users toward completing high-priority tasks before deadlines are missed.

---

## Key Features 🚀

### Intelligent Risk Analysis

Each task is assigned a dynamic risk percentage based on urgency, importance, deadlines, and completion behavior.

### AI Action Plan & Suggested Order

Gemini generates personalized daily productivity insights and recommends the best execution order for tasks.

### Multiple AI Guardian Personalities

Users can choose their preferred AI coaching style:

* **Ava — Supportive Friend**
* **Strict Coach**
* **Calm Mentor**
* **Chronos — Competitive Rival**

Each personality delivers a unique motivational experience while maintaining accountability.

### Focus Mode

A distraction-free productivity mode featuring:

* Timer presets
* Custom durations
* XP rewards
* Badge unlocks

### Proof-Based Task Completion

Instead of blindly marking tasks complete, the system requests proof such as:

* Progress snapshots
* Contextual validation
* Completion confirmation

This encourages honest productivity tracking.

### Chronos Rival System

A gamified AI rival that gains XP daily, motivating users to remain productive and outperform their AI competitor.

### Responsive Cross-Platform Design

Optimized for:

* Desktop
* Tablet
* Mobile

## Run Locally

### Prerequisites

* Node.js
* Gemini API Key

### Setup

Install dependencies:

```bash
npm install
```

Set the `GEMINI_API_KEY` in `.env.local`:

```env
GEMINI_API_KEY=your_api_key_here
```

Run the application:

```bash
npm run dev
```
